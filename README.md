## ダウンロード
[epcviewer](https://github.com/mrhorin/epcviewer/releases)

## 特徴
- クロスプラットフォーム
- ダーク/ライトテーマの切り替え機能
- タブ表示機能
- OBS等のブラウザソースから字幕を表示できる

## 使い方
[epcyp](https://github.com/mrhorin/epcyp)などのYPブラウザにBBSブラウザとして登録して使用
